# Global Consulting Group — Web Sitesi

Antalya merkezli tercüme, noter işlemleri ve ikamet/göç danışmanlığı hizmetleri sunan Global Consulting Group için kurumsal tanıtım sitesi.

Next.js (App Router), TypeScript, Tailwind CSS ve Framer Motion ile geliştirilmiştir.

## Geliştirme

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) adresinden görüntüleyebilirsiniz.

## Yapı

- `src/app` — sayfa ve genel layout
- `src/components` — Header, Hero, About, Services, Testimonials, Contact, Footer, WhatsAppFab

## Deploy

Proje [Vercel](https://vercel.com) üzerinde barındırmak için hazırdır — GitHub reposunu Vercel'e bağlamanız yeterlidir.
